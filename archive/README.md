# Archive

This directory stores files removed from the active H10 working surface.

- `legacy_manuscripts/` contains earlier manuscript drafts, generated preview
  PDFs, LaTeX auxiliary files, and old source tarballs.
- `legacy_notes/` contains stale hand-written Lean issue locks, punchlists, and
  reconnaissance notes that are no longer the active source of truth.

The canonical manuscript is now the root-level
`semantic_convergence_interactive_learning.tex`. The active completion list is
`h10_lean_correspondence_punchlist.md`.
